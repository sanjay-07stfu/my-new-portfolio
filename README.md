# My New Animated Portfolio

A modern, animated personal portfolio website built to showcase skills, projects, and experience with smooth animations and an engaging user interface.

## Features

- ✨ Smooth CSS/JS animations throughout the site
- 🎨 Clean, modern UI design
- 📱 Fully responsive layout for all screen sizes
- 🚀 Fast and lightweight
- 💼 Sections for About, Skills, Projects, and Contact

## Tech Stack

- HTML5
- CSS3 (Animations & Transitions)
- JavaScript

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/sanjay-07stfu/my-new-portfolio.git
   ```
2. Open `index.html` in your browser to view the portfolio.
